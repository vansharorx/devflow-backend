# Devflow
Enterprise-grade Issue Tracking System built using Node.js, Express.js, MySQL, JWT Authentication, Docker, Socket.IO and GitHub Actions.

## Features

- JWT Authentication
- Role Based Access Control
- Projects Management
- Issue Tracking
- Issue Comments
- Activity Timeline
- Notifications
- Dashboard Analytics
- Soft Delete & Restore
- File Uploads
- Email Notifications
- WebSockets
- Docker Support
- CI/CD Pipeline
- Rate Limiting
- Health Monitoring


## Architecture

Client
   ↓
REST API
   ↓
Controllers
   ↓
Services
   ↓
Models
   ↓
MySQL
